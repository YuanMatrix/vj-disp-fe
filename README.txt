# 🎬 VJ Display - 快速开始

## ⚡ 最简单的方式

**直接双击这个文件：**

```

```

这个脚本会：
1. ✅ 自动检查 Node.js
2. ✅ 自动安装依赖（首次运行）
3. ✅ 让您选择启动模式
4. ✅ 自动打开浏览器

## ❓ 常见问题

### Q: 双击后闪退？
**A:** 可能没有安装 Node.js，请先安装：
- 下载地址：https://nodejs.org/
- 选择 LTS 版本（长期支持版）

### Q: 端口被占用？
**A:** 3000 端口被其他程序占用
- 关闭占用 3000 端口的程序
- 或修改端口（见详细文档）

### Q: 如何停止服务器？
**A:** 在命令行窗口按 `Ctrl + C`
